Translit Alfred 3 workflow

Posted at — Jul 14, 2017

This is Alfred “translit” workflow. It fix mistakes when entering cyrillic text in wrong input locale (english) and automatically switches keyboard layout. In short it does the same as the Punto Swither does.

To work correctly you need input source switcher and Python3.

To install workflow in Alfred, download latest release an import it by clicking in Finder.
